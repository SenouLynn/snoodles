# Snoodles

Personal Claude Code plugin. Successor to [snood](https://github.com/senoulynn/snood).

## Setup

**Prerequisites:** [Claude Code](https://docs.anthropic.com/en/docs/claude-code) CLI installed.

**Install:**

```bash
git clone <repo-url> snoodles
claude plugin marketplace add snoodles /path/to/snoodles
claude plugin install snoodles
```

**Uninstall:**

```bash
claude plugin uninstall snoodles
```

## Commands

| Command | Purpose |
|---------|---------|
| `/snoodles:derive-prompt` | Write, fix, or improve prompts for any AI tool |
